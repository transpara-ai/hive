### Detail

**`DelegationRow` + `GetUserDelegation` + `ListDelegations` (store.go)**
- `GetUserDelegation(ctx, spaceID, actorID)` returns delegate ID + name for the current user; used to show "Delegated to X" or the delegation form
- `ListDelegations(ctx, spaceID, limit)` lists all space delegations with names resolved via LEFT JOIN on space_members; BOUNDED at limit

**`handleGovernance` (handlers.go)**
- Calls `GetUserDelegation` for the logged-in user
- Calls `ListDelegations` for the space (limit 20)
- Passes `hasDelegated`, `delegateName`, `delegations` to `GovernanceView`

**`GovernanceView` (views.templ)**
- **Delegation panel**: visible when user is logged in. Shows "Delegated to X + Undelegate button" or "Delegate vote form". Includes collapsible list of all active space delegations.
- **Proposal creation form**: adds `quorum_pct` (number 0-100) and `voting_body` (all/council/team) fields
- **Proposal card**: adds quorum indicator (`quorum 51% · council`) when `QuorumPct > 0`
